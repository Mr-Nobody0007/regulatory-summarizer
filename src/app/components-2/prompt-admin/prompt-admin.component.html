Prompt Types and Purposes

  • Default Prompts: Each document type (e.g., "Notice", "Rule") must have exactly one prompt with the purpose "Default". This is used for initial document summarization.
  • Suggestion Prompts: Each document type can have multiple prompts with the purpose "Suggestion". These appear as suggested prompts in the chat interface.
  • NoDocDefault: There must be exactly one prompt with both type and purpose set to "NoDocDefault". This is used when a document type isn't available (like URL inputs).

Writing Effective Prompts

  • Be specific about what information the AI should extract or analyze.
  • For Default prompts, focus on comprehensive summaries that highlight key requirements and deadlines.
  • For Suggestion prompts, focus on specific aspects (deadlines, requirements, implications) to help users get targeted information.
